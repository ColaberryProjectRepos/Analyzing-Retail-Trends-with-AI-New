Building Datawarehouse in Microsoft Fabric 11

1. Creating Datawarehouse in Microsoft Fabric

Create a new workspace in Microsoft fabric and name it.


Within your created workspace, create a new warehouse to store and manage your data


Create a table in the Data Warehouse.

To Load files into Microsoft Fabric Datawarehouse, Follow these steps:

  • Create a new Lakehouse and upload the Dataset





Once the file is loaded in Lakehouse, add new pipeline to perform loading data into Datawarehouse from Lakehouse. Name the pipeline as LHtoDH


Configure the source and destination.






  • Perform column mappings.
  • Ensure that the data types are matching between source and destination. 
  • Run the pipeline to ensure successful execution.

    •  

    The data has been successfully loaded into the Data Warehouse from the Lakehouse.
