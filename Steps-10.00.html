2. Getting code in Jupyter notebook 

After clicking on view code in Nexusmax, the code will be displayed in a Jupyter Notebook.


3. Convert this code to spark/python to make it compatible for Microsoft Fabric

Use ChatGPT to convert Jupyter Notebook code to Spark/Python code




Repeat step 3 to convert all the codes in jupyter notebook to spark/python one at a time
