Data Overview 2
Data Sources
The data was sourced from Kaggle and includes comprehensive retail customer records, covering transaction histories, customer demographics, interaction data, product details, facilitating detailed studies and insights.

 
System Architecture
The architecture integrates data ingestion modules, preprocessing pipelines, and machine learning workflows, all hosted on a cloud platform with Microsoft Fabric Data Warehouse. This setup ensures scalability, real-time data processing, and efficient data storage.

Data Features
Key features included customer transaction history, demographics, and interaction data. The data dictionary highlighted critical KPIs used to measure churn and retention rate.
